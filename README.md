# 🏠 Muvit

**Muvit** es una web app estilo "Tinder para propiedades" para el mercado inmobiliario de Buenos Aires.

## Features
- 📱 Swipe de propiedades (izquierda / derecha)
- 🤖 Chatbot de IA por propiedad
- 📊 Dashboard de propietario con métricas
- 📄 Pantalla de detalle con amenities y contacto

## Stack
- React + Vite
- CSS-in-JS (inline styles)
- SVG ilustraciones (sin dependencias externas)

## Desarrollo local

```bash
npm install
npm run dev
```

## Deploy en Vercel

1. Push este repo a GitHub
2. Importar en [vercel.com](https://vercel.com)
3. Vercel detecta Vite automáticamente → Deploy
